# Quick Installation Guide

This is the minimal setup guide to get infrastructure running on local machine.

## Prerequisites

You must have the following installed on your host machine:
- **Git**
- **Docker & Docker Compose** (for running GitLab CE)
- **VirtualBox** (for the VM)
- **Vagrant** (to automate VirtualBox)

*(Note: Ansible is automatically installed inside the VM, you do NOT need it on your host but can install it on your host machine by installing python3 and pip3 and then `pip3 install ansible`)
